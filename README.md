# СУБД Oracle

В папке Oracle есть файлы Task_*.txt их 6 штук, как и пунктов к заданию (само задание можно просмотреть в TASKS.md). 

Эти файлы представляют собой документацию, в этих 6 файлах будет сказано что и зачем было закинуто в этот репозиторий. 

Поэтому, если теряетесь в догадках зачем какой-то из файлов, то просто просмотрите Task_*.txt.
