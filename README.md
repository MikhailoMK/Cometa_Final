# СУБД Oracle

В папке Oracle есть файлы Task_*.txt их 6 штук, как и пунктов к заданию. 

Эти файлы представляют собой документацию ко второй части. В этих 6 файлах будет сказано что и зачем было закинуто в папку. 

Поэтому, если теряетесь в догадках зачем какой-то из файлов, то просто просмотрите Task_*.txt.
